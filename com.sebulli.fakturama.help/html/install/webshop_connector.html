Die Schnittstelle zum Webshop

Damit Fakturama Bestellungen aus dem Webshop abrufen kann ist es notwendig, auf dem Server des Shopsystems noch die Webshop Schnittstelle zu installieren.

Download

Im Downloadbereich auf der Fakturama Projektseite finden sich Schnittstellen zu unterschiedlichen Shopsystemen. Entpacken Sie das entsprechende ZIP Archiv auf Ihrem Rechner. Es müssten sich folgende drei Dateien im entpackten Ordner befinden:


Die Datei INSTALLATION.TXT beschreibt die Installation. Die beiden Dateien fakturama_connector_settings.php und fakturama_connector.php müssen auf den Webserver kopiert werden. Vorher ist die Datei fakturama_connector_settings.php noch anzupassen.
