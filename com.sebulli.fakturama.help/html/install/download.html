Download der Software

Fakturama steht auf der Projektseite zum Download zur Verfügung.

Das Programm ist kostenlos, OpenSource und steht unter der Lizenz EPL v1.0

Fakturama läuft unter folgenden Betriebssystemen:

  • Windows XP, Windows Vista, Windows 7 (32Bit und 64Bit)
  • Mac OS X
  • Linux (32Bit und 64Bit)

Zur Ausführung ist OpenOffice und Java™ erforderlich.

  • Downloadseite OpenOffice: http://de.openoffice.org/
  • Downloadseite Java™: http://www.java.com/de/
  • Überprüfen, ob Java™ bereits installiert ist: TEST

Wird Fakturama zusammen mit einem Webshop betrieben, ist zusätzlich die entsprechende Webshop Schnittstelle zu installieren. Diese findet sich ebenfalls auf der Projektseite
