################ INSTALLATION ##############
Copy the files
- fakturama_connector.php
- fakturama_connector_settings.php
into the folder /admin/ in your web shop.

Edit fakturama_connector_settings.php and ..

- Set the shop system
- Choose a user name and password.
- Set your language and your country.
